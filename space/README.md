# Text-to-SQL — Mistral-7B fine-tuned with QLoRA

Plain-English question + table schema → SQL query, generated by a Mistral-7B model
fine-tuned with QLoRA and served as a 4-bit GGUF on a free CPU Space.

- **Training notebook + writeup:** [github.com/Nagendra7299/text-to-sql-lora](https://github.com/Nagendra7299/text-to-sql-lora)
- **Model:** [Nagendra729/mistral-7b-text-to-sql-gguf](https://huggingface.co/Nagendra729/mistral-7b-text-to-sql-gguf)

## How it works
The app auto-discovers the `.gguf` file in the model repo, downloads it on first boot,
and runs inference with `llama-cpp-python`. Override the source with the `GGUF_REPO`
environment variable.

> Note: first response after a cold start is slower while the model file downloads and loads.
