# Assets

Drop `demo.gif` here (a screen recording of the model converting English → SQL),
then uncomment the image line in the root `README.md` Demo section.

How to make it:
1. Run the "Try it yourself" snippet in Colab (T4 GPU)
2. Screen-record it generating SQL for 2-3 questions
3. Convert the recording to GIF (e.g. https://ezgif.com/video-to-gif)
4. Save as `assets/demo.gif`
